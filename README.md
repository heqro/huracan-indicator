# Requires

* Latte >= v.0.10.4

# Install

From Latte UI: **Effects -> Indicators -> Style -> Get New Indicators...**

From command line:

```kpackagetool5 -i . -t Latte/Indicator```

# Update

From Latte UI: **Effects -> Indicators -> Style -> Get New Indicators... -> Installed**

From command line:

```kpackagetool5 -u . -t Latte/Indicator```
